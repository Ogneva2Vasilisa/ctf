# Проклятый старый сайт

*На стареньком сайте семейной мастерской поселился вирус. Он пугает посетителей: рушит вёрстку, меняет местами товары, переписывает ценники, ругается через онлайн-консультанта. Найдите его и изгоните.*

Для решения сначала можно отдебажить этот .js в браузере, посмотреть, по какой очереди выполняются функции. Далее подбираем красивый ключ по каждому шагу:

1. Проверка, что все поля заполнены верно и 4 тире
2. Проверка последних 4 значений - контрольная сумма, генерируем для своего ключа на [сайте для отлаживания своих функций на javascript](https://developer.mozilla.org/ru/docs/Web/JavaScript/Reference/Statements/for) 
3. Проверка на то, что первые 8 значений ключа содержат только символы из [A-F0-9]{8}
4. Проверка, что 2 значения из 8-10 позиций проходят по функции (на сайтике, упомянутом выше отлаживаем)

Вводим ключ - флаг наш
